# VaultX - Personal Finance Dashboard

A modern, feature-rich finance tracking application built with React. Track your income, expenses, set budgets, and get insights into your spending habits.

## Features

- 📊 **Dashboard Overview** - See your financial summary at a glance
- 💰 **Transaction Management** - Add, edit, and delete transactions easily
- 🎯 **Budget Tracking** - Set monthly budgets and monitor your spending
- 📈 **Visual Analytics** - Multiple chart types for better insights
- 📥 **Export Data** - Download your data as CSV or PDF
- ⌨️ **Keyboard Shortcuts** - Navigate quickly with hotkeys
- 🌙 **Dark Mode** - Easy on the eyes
- 📱 **Responsive Design** - Works on all devices

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion (animations)
- Recharts (data visualization)
- React Router
- date-fns

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Keyboard Shortcuts

- `1`, `2`, `3` - Navigate between pages
- `Ctrl/Cmd + N` - New transaction
- `Ctrl/Cmd + E` - Export data
- `Ctrl/Cmd + T` - Toggle theme
- `?` - Show all shortcuts
- `Esc` - Close modals
