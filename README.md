# 🚀 VaultX – Finance Dashboard

## 📌 Overview
A modern finance dashboard built to simulate real-world financial tracking with clean UI and insightful analytics.

## ✨ Features
- Financial summary cards
- Interactive charts
- Transaction management
- Role-based UI (Admin/Viewer)
- Smart insights engine
- Dark mode
- Local storage persistence

## 🧠 Approach
Focused on:
- Clean UX
- Modular architecture
- Real-world usability
- Scalable state management

## 🛠 Tech Stack
- React
- Tailwind CSS
- Recharts
- Context API

## ⚙️ Setup
```bash
npm install
npm run dev
```

## 🎯 Key Highlight
Instead of just displaying data, this dashboard derives meaningful insights to help users understand spending behavior.
